# 🛒 E-Commerce Sales Analysis Dashboard

> End-to-end Data Analysis project built with Python, Pandas, Matplotlib, Seaborn, and Power BI  
> Analyzing 5,500+ e-commerce transactions to uncover sales performance, customer behavior, and regional profitability insights.

---

## 📌 Project Overview

An Indian e-commerce company sells products across 6 categories and 4 regions.  
Management needed answers to:
- Which products and categories generate the most revenue and profit?
- How do sales trend across months and seasons?
- Which regions and customer segments perform best?
- Why is the return/cancellation rate so high?

This project answers all of the above with clean data, professional visualizations, and actionable business recommendations.
